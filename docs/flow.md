# GitStar AI Flow

```text
Goal
  -> Loop Pattern
  -> Skill
  -> STATE.md Memory
  -> Agent Execution
  -> Verification
  -> Human Review
  -> Action
  -> Run Log
  -> Updated STATE.md
```

## Flow

1. Define the goal.
2. Select a loop pattern.
3. Load the matching skill.
4. Read project memory from STATE.md.
5. Run the agent task.
6. Verify output.
7. Ask for approval when needed.
8. Update the run log.
