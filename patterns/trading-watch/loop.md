# Trading Watch Loop

## Purpose
Create educational market-watch summaries and risk-defined setup cards.

## Inputs
- Market price data
- Chart levels
- Volume data
- News context
- User-defined watchlist

## Steps
1. Read watchlist.
2. Pull current price and relevant levels.
3. Check trend, support, resistance, volume, and invalidation.
4. Create an educational setup card.
5. Mark status as Watch, Confirmed, Invalid, or Review.

## Output
Educational market-watch card, not financial advice.

## Safety
- No guaranteed profit language.
- No direct trade instruction.
- Always include invalidation and risk note.
- Human approval required before posting.
