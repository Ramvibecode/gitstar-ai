# Dashboard Demo

This folder contains a static dashboard concept for GitStar AI.

## Run Locally

```bash
cd demo
python -m http.server 8080
```

Open:

```text
http://localhost:8080
```

## Purpose

The dashboard shows how GitStar AI can present:

- active loops
- skill packs
- project state
- human approval mode
- loop pipeline

This is only a frontend concept for now. Future versions can connect to real loop run logs and state files.
